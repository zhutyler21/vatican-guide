# 面向中国用户的网络加速说明

本项目已经做了前端侧弱网优化：

1. 首屏只预缓存核心文件，避免第一次打开就拉取所有图片/音频。
2. 页面顶部新增「中国用户加速模式」：用户可以一键缓存全部图片和语音。
3. Service Worker 支持图片、音频运行时缓存；缓存后馆内弱网/无网也能听。
4. Cloudflare Pages `_headers` 已给图片/音频设置浏览器/CDN 缓存。

注意：`https://vatican-guide.pages.dev/` 的首次访问速度取决于 Cloudflare Pages 在中国大陆的连通性。
如果用户在国内连 pages.dev 都打不开，前端代码无法执行，也就谈不上“代码加速”。真正稳定的大陆访问，需要：

- 绑定自有域名；
- 做 ICP 备案；
- 接入阿里云/腾讯云/华为云/又拍云等国内 CDN 或静态托管；
- 把本仓库静态文件同步到该 CDN 源站。

当前改动解决的是「能打开以后更快、更稳、可离线」。
